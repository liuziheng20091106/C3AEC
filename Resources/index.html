# 资源提交

我们十分欢迎您提交资源到3班学术交流中心，提交方法如下：

  • 电子邮件：发送资源详细到tkoffice@foxmail.com，我们看到后会尽快回复（最方便）
  • QQ/TIM联系：加入3班学术交流群，然后向管理员小窗发送资源详细（最快回复）
  • 微信联系：（不建议）加入3班学术交流群-微信班，或者寻找微信号liuziheng20091106并添加好友，发送资源详细
  • 短信：发送资源详细到13964585619(有短信费哦~)

# 资源详细格式

我是:[姓名]
我申请提交 学术资源

[资源链接 / 附件 / 其他形式...]
