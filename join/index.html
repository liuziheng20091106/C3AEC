# 加入3班学术交流群

# 请注意，即使你没有加入聊群，你仍可以在3班学术服务中心享受类似的服务

# 加群条件：

  • 你是山东省烟台市第二中学-莱山校区-初2021级3班的学生
  • 你的学习成绩在班级前30
  • 你一个月至少能用一次手机等聊天工具
  • 你不是话痨用户，也不是潜水用户
  • 你是一个**有信用的人

# 加群途径：

  1. 通过群内成员拉你入群
  2. 通过与管理员交流入群
  3. 在官网申请入群（不建议）
