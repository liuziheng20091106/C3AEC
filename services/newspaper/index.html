# 3班月报

# 这是什么？

3班月报是由3班学术交流群成员收集、整理而成的一月学校报告。

# 如何参加？

首先你得是3班学术交流群成员。

你只需要在月天以前将你所知道的消息发送给群主，如果内容合理，将会被添加进当期月报里。

# 更新规则？

  • 正常时候是1月一更。
  • 假期时间不更新
  • 短期节假日（3天-7天）将根据情况更新
  • 考试复习阶段不更新
  • 大考后立即更新
  • 没有足够题材时不会更新
  • 其他情况另有说明
